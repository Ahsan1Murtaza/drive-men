# File Upload and Download Web App

This project is a simple web application that allows users to upload files and later download them using their original filenames. It provides an easy-to-use interface for managing files on the server.

---

## 🚀 Features
- Upload files from the browser
- Display a list of uploaded files
- Download files using their original names

---

## 📂 How It Works
1. Select a file and upload it through the web interface.
2. The application stores the file on the server and keeps track of its original name.
3. Uploaded files are displayed in a list with download links.
4. When you download a file, it is downloaded with its original name.
